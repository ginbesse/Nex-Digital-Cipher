# Kullanım Kılavuzu

## Hızlı başlangıç

```bash
python3 smart_device_generator.py
```

## Demo üretimi

```bash
python3 smart_device_generator.py --demo
```

## Termux kurulumu

```bash
sh termux_setup.sh
```

## Desteklenen türler

- Website
- Airplane
- Car
- Ship
- Phone
- Computer
- Digital App

## Notlar

- Website seçimi için domain ve URL girişi isteyebilirsiniz.
- Her çıktı, kullanıcı sayısı ve metadata ile birlikte üretilir.
- Üretilen dosyalar `generated/` altında saklanır.
